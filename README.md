# CryptoSim Move (Upgraded Flutter)
Features:
- Multi-symbol: BTCUSDT, ETHUSDT, BNBUSDT, XRPUSDT (selectable)
- Multi-timeframe charts: 15m, 1h, 4h
- Indicators: EMA, RSI, MACD, ATR
- Buy/Sell markers on charts
- Virtual wallet: 1,000 USDT starting balance, 10% per trade
- Local persistence: save/load wallet and trade history
- Dashboard, Chart, Settings, Trade History screens
- Debug APK ready to build
